## Deploy to aiwithblake.com

### Option 1: Netlify (Easiest)
1. Go to netlify.com
2. Drag and drop the aiwithblake folder
3. Connect your domain
4. Done

### Option 2: Vercel
1. Install Vercel CLI: `npm i -g vercel`
2. Run: `vercel` in the aiwithblake folder
3. Connect your domain
4. Done
